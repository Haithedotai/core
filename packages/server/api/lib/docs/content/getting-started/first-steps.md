# First Steps

Get started with Haithe in just a few simple steps.

## 1. Connect Your Wallet

The first step is to connect your Web3 wallet to Haithe:

1. Click the **"Connect Wallet"** button in the top navigation
2. Choose your wallet provider (MetaMask, WalletConnect, etc.)
3. Approve the connection in your wallet
4. Sign the authentication message

Your wallet address will now appear in the navigation bar.

## 2. Create an Organization (Optional)

Organizations allow you to collaborate with team members:

1. Go to the **Dashboard**
2. Click **"Create Organization"**
3. Enter a name and description
4. Click **"Create"**
5. Invite team members using their wallet addresses

## 4. Explore the Marketplace

Discover AI agents and tools:

1. Click **"Marketplace"** in the navigation
2. Browse available AI agents and tools
3. Click on any item to learn more
4. Try interacting with free or trial options

## 5. Create Your First Agent

Ready to build something custom?

1. Go to **Agents** in the dashboard
2. Click **"Create New Agent"**
3. Choose a template or start from scratch
4. Configure your agent's personality and capabilities
5. Deploy and start chatting!
